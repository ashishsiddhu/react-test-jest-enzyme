# react-test-jest-enzyme

## Features

- React
- Testing
  - Jest + Enzyme: Unit/Integration + Snapshot Test
  
## Installation

- `git clone https://github.com/ashishsiddhu/react-test-jest-enzyme.git`
- cd react-test-jest-enzyme
- npm install
- npm start
- visit `http://localhost:3000/`
- npm run test
